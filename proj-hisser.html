Project - Hisser

A twitter clone created using PHP, MySQLi, HTML, Bootstrap, and CSS

Live Demo

Background

This project was an attempt to build all of Twitter's basic functionalities using PHP, Bootstrap, HTML, CSS, and MySQLi. Currently, it supports custom profiles, user feeds, likes, and authentication.

Sample Code

This code gets all necessary information through MySQLi queries to create a "feed" for the logged in user in the current session.

public function getFeed() {
    if($_SESSION["userid"] == $this->userid) {
      echo '<div class="panel-heading">Your feed<div>';
      
      $followingSql = "SELECT following_userid FROM followers WHERE follower_userid=".$this->userid;
      $sql = "SELECT * FROM comments 
              INNER JOIN users ON comments.userId = users.userId 
              WHERE comments.userid=".$this->userid." OR comments.userid IN (".$followingSql.") ORDER BY commentdate DESC";
      
      $result = $this->conn->query($sql);
      $rowcount=mysqli_num_rows($result);
      if ($rowcount > 0) {
        echo '<div class="panel-body" style="word-wrap: break-word">';
        while($row = $result->fetch_assoc()) {
        
        //render profile pics

        $sqlImg = "SELECT * FROM profileimg WHERE userid = ".$row['userid'];
