Project - React Portfolio

An online portfolio app created using React, Node, JavaScript, HTML, CSS, and React Bootstrap

My Dad's Portfolio - Live Demo

Background

My father, Jim Teller, wanted to have a personal IT portfolio online. I offered my help and decided to use React, which allowed me to rapidly develop a template application based on components that anyone could use to quickly launch an online portfolio. This open source React project is available on my GitHub and was used to create a new revision of my portfolio that I plan on publishing in the future, along with my Dad's online portfolio he is using currently. His portfolio was published to Github Pages.
